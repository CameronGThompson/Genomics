# Genomics

This repository contains shell scripts for phylogenetic tree building workflows, genome assembly and annotation workflows, and RNA-seq and Differentially Expressed Genes (DEGs) workflows. These are based off of practicals we did for our Genomics module.

There are 5 scripts in this repository: the phylogenetics script (phylogenetic_trees.sh), the genome assembly/annotation script (assembly_and_annotation.sh), the RNA-seq script (RNA-seq.sh), the DEG script (DEGs.R), and the functional annotation script (functional_annotation.sh). 

Analyses were performed originally on a High Performance Computer at the university.
